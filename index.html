Emolock

Unlock with Emotion

The EmoLock is an emotion-controlled lock system implemented in Python and C. It uses a predefined sequence of emotions as the unlock password. A Logitech USB camera captures facial expressions, which are processed by the computer backend. An Arduino Mega is then programmed to manage the automatic locking and unlocking of a solenoid electromagnetic cabinet lock.

Software: Python • OpenCV • Deepface • C

Hardware: Solenoid electromagnetic lock • Arduino Mega
